Functional Languages

Functional programming is a completely different beast from OOP languages because it uses functions to accomplish its goals. Some argue it is more difficult to learn than OOP, but it's just a very powerful and concise way of solving problems. Some, if not all, of the OOP programming languages have functional capability like Lambda functions.
