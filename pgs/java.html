JAVA

This is all about Java. The most ubiquitous programming language in the world!

Regardless of how old, verbose or tough to learn this language is, it remains the foundational language for most software out there. It is heavily used in enterprise environments, android development, game dev, and back end development.

It is also the most in demand skill in the job market today. Well beyond any other language. (see stats below)

It is a purely object oriented language that heavily borrows from C++ and C. Not to be confused with or equated to JavaScript, which is a much lighter, and front end based programming language for the web.
